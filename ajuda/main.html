Bem-vindo à Ajuda do DentalFX

Aqui você encontra guias rápidos sobre como usar as principais funções do sistema. Navegue pelos tópicos na barra lateral à esquerda.

Cadastrar uma nova informação (função da secretária/administrador):

O sistema permite cadastrar Pacientes, Dentistas, Usuários do sistema, Materiais e Procedimentos. Os passos são similares para todos:

  1. Na tela principal (Agendamentos), localize o menu "Cadastros" ou "Sistema" na barra de menu superior.
  2. Clique na opção desejada (ex: "Pacientes", "Dentistas", "Materiais").
  3. Uma nova janela de gerenciamento será aberta. Clique no botão "+".
  4. Preencha o formulário com as informações solicitadas.
  5. Clique em "Confirmar" para gravar os dados.

Para **alterar** ou **apagar** um registro, basta clicar nele com o botão direito do mouse e clicar no botão "Alterar" ou "Apagar".

Agendando uma consulta

Para agendar um paciente (função da secretária/administrador):

  1. Na tela principal de "Agendamentos".
  2. Selecione o Dentista desejado no menu suspenso.
  3. Escolha a Data da consulta no calendário.
  4. A tabela de horários do dia será carregada.
  5. Clique em um horário que esteja marcado como "Livre".
  6. Clique no botão "Agendar".
  7. Uma nova janela se abrirá. Nela, selecione o paciente.
  8. Clique em "Confirmar" para finalizar o agendamento. O nome do paciente aparecerá no horário.

Atenção: Você não pode agendar em um horário que já esteja ocupado ou em uma data que já passou.

Cancelando uma consulta

Para cancelar um agendamento existente (função da secretária/administrador):

  1. Na tela principal de "Agendamentos".
  2. Selecione o Dentista e a Data da consulta que deseja cancelar.
  3. A tabela de horários será carregada.
  4. Clique no horário ocupado (com o nome do paciente) que você deseja cancelar.
  5. Clique no botão "Cancelar Agendamento".
  6. O agendamento será removido e o horário voltará a ficar "Livre".

O Dentista também pode cancelar um agendamento pela sua própria tela de atendimento, seguindo um fluxo parecido.

Registrando o atendimento (Visão do Dentista)

Esta função é realizada pelo Dentista após o login, na sua tela de atendimento:

  1. O dentista deve selecionar a Data do atendimento no calendário.
  2. A sua agenda do dia aparecerá na tabela "Horários".
  3. Ao clicar em um horário com paciente, o sistema habilitará os campos de atendimento.
  4. O histórico do paciente será carregado no campo de texto principal.
  5. O dentista pode adicionar Materiais e Procedimentos usando os menus e os botões "+".
  6. Os itens adicionados aparecerão nas tabelas de "Materiais Utilizados" e "Procedimentos Realizados".
  7. O dentista deve escrever o Relato do Atendimento no campo de texto (isso substituirá o histórico que foi carregado apenas para visualização).
  8. Ao finalizar, o dentista deve clicar no botão "Efetivar".

Uma vez efetivado, o atendimento é salvo e não pode mais ser editado. Os materiais e procedimentos são registrados para o relatório financeiro.

Como fazer backup e restaurar os dados

Importante: Esta é uma função crítica e só está disponível para usuários com nível de **Administrador**.

Para fazer um Backup:

  1. Na tela principal, acesse o menu "Sistema" na barra de menu superior.
  2. Clique em "Fazer Backup".
  3. O sistema criará um arquivo (ex: sisdental.sql) na pasta bdutil/backup/, dentro da pasta onde o programa está instalado.
  4. Recomendação: Copie este arquivo para um local seguro (pen drive, Google Drive, e-mail) regularmente.

Para Restaurar um Backup:

  1. Verifique se o seu arquivo de backup (ex: sisdental.sql) está dentro da pasta bdutil/backup/.
  2. Na tela principal, acesse o menu "Sistema".
  3. Clique em "Restaurar Backup".
  4. Atenção: O sistema irá apagar TODOS os dados atuais e substituí-los pelos dados do arquivo de backup. Esta ação não pode ser desfeita.
